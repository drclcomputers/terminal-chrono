# terminal-chrono

Complete clock app for the terminal, including features such as:
-clock<br>
-calendar<br>
-chronometer<br>
-timer<br>
<br><br>
For help, type the programs name in the terminal.
<br><br>
How to use:<br>
Download the exe for Windows 64bit.<br>
Compile "terminal-chrono.cpp" with g++.
